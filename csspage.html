CSS Selectors Example Description
element a Selects all <a> elements.
#id #name Selects the element with id="name".
* * Selects all elements.
.class .board
p.board		 Selects all elements with class="board". Selects all <p> elements with class="intro".
CSS BOX MODEL Description
Content The content of the box, which includes text and images.
Padding Padding refers to the transparent space between the element’s content and its border. The background color of an element will also be seen through its padding area if it has one.
Border The padding and content are surrounded by a border.
Margin Margin refers to the transparent area around the border. The margin area is always transparent; it is unaffected by the element’s background color, but it allows the parent element’s background color to be seen through it.
CSS DISPLAY Description
inline Displays an element as an inline element.
block Displays an element as a block element.
flex Displays an element as a block-level flex container.
grid Displays an element as a block-level grid container.
inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values.
inline-flex Displays an element as an inline-level flex container.
